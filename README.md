* Пример данных

Есть фича SMILES которая показывает структуру\связи молекулы, потом идут более 400 числовых фич.
Числовые фичи показывают различные диапозоны возможных значений:
![alt text](image-1.png)
![alt text](image-2.png)

Дальнейший анализ показал, что большинство таких, у которых 2-3 различных значений. И также было замеченно, 
что есть некоторые, которые имеют все одинаковые занчения! Как например группа 90. Много и таких, у которых все нули кроме 1-20 значений.
Такие фичи будут бесполезны.

Я в дальнейшем буду производит отбор фич через оценку их важности в модели. 

Для обработки SMILES была импортированна библиотека rdkit которая составляет граф молекулы, где:
- атомы - вершины
- связи - ребра
- типы связей, и другие характеристики - свойства узлов

Потом библиотека сроит множество подграфов в зависимости от radius. 

Каждый подграф мы хешируем получаем одну единицу, хакактеризующую существование в молекуле данного подграфа (структуры)
и получаем общий хеш, который будет отражать паттерн молекулы. От этого модель будет улавливать зависимость
что это за молекула.
